### Random useful stuff
- macOS zip and encrypt: `zip -e <ARCHIVE_NAME> <TARGET>`
- Create SSH key pair: `ssh-keygen -t ed25519 (optional: -C "email@address.example")` or `ssh-keygen -t rsa -b 4096 (optional: -C "email@address.example")`
